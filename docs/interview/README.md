## Quick Reference — Key Topics Per File

### 1. Backend Architecture
Monolith vs Microservices, Distributed Transactions (Saga), API Versioning, CAP Theorem, Rate Limiting, CQRS & Event Sourcing, Notification System Design, Graceful Degradation

### 2. Spring Boot
DI & Stereotypes, Auto-configuration, Security & JWT, Actuator, DB Migrations (Flyway), Bean Lifecycle, Profiles, @Transactional Pitfalls, MVC vs WebFlux, Spring Cloud, Caching, AOP, @Async, Request Annotations

### 3. Database
Clustered vs Non-Clustered Indexes, Query Optimization (EXPLAIN ANALYZE), SQL vs NoSQL, Connection Pooling (HikariCP), N+1 Problem, Transaction Isolation Levels, Database Sharding, Replication (Master-Slave)

### 4. DevOps
CI/CD Pipelines, Docker Best Practices, Kubernetes Concepts, Secrets Management, Observability (Logs/Metrics/Traces), Blue-Green & Canary Deployments, Log Aggregation, Infrastructure as Code

### 5. AI Integration
LLM API Integration, RAG Pattern, Hallucination Mitigation, Cost Optimization, SSE Streaming, Document Processing Pipeline

### 6. Unit Testing
Testing Pyramid, Slice Tests (@WebMvcTest, @DataJpaTest), @Mock vs @MockBean, Testable Code Design, Async Testing (Awaitility), Testcontainers, Contract Testing (Pact), Mutation Testing (PIT)

### 7. DSA
Array vs LinkedList, HashMap Internals, Sorting Algorithms, Trees & Graphs (BFS/DFS), Stack & Queue, BST, Dynamic Programming, Two Pointers, Big O, Reverse LinkedList, Valid Parentheses, Dijkstra's Algorithm, Recursion vs Iteration
